# Random Linear Regression (from Scratch)

This project implements **linear regression using gradient descent** without scikit-learn.

# Dataset
- Source: [Random Linear Regression Dataset on Kaggle](https://www.kaggle.com/datasets/andonians/random-linear-regression)

# Implementation Steps
1. Loaded and visualized data
2. Implemented cost function
3. Computed gradients manually
4. Trained using gradient descent
5. Predicted values for new inputs
